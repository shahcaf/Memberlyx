# Memebrlxy

Official website for the Memebrlxy Discord bot — community growth and server management.

## Local preview

```bash
npx serve . -l 3456
```

Open [http://localhost:3456](http://localhost:3456).

## Deploy

This repo is a static site. Host on GitHub Pages, Netlify, Vercel, or Cloudflare Pages with no build step.

### GitHub Pages

1. Go to **Settings → Pages**
2. Source: **Deploy from a branch**
3. Branch: `main` / `/ (root)`
4. Save — your site will be live at `https://shahcaf.github.io/Memberlyx/`
